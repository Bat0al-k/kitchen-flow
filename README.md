## 🍽️ KitchenFlow — Restaurant Order Management System (Kitchen View)

KitchenFlow is a simple MVP web application built with Next.js App Router, designed to help restaurant kitchens manage incoming orders efficiently.

The system provides a clear kitchen dashboard where staff can:

Create new orders with table number and items

View all active orders in real-time order flow

Update order status through different stages:
Pending → Cooking → Ready

Delete orders when necessary (before preparation)

This project focuses on practicing modern Next.js full-stack concepts such as:

Server Components

Server Actions (without traditional REST APIs)

Dynamic Routing

Data fetching and revalidation

Clean scalable folder structure

KitchenFlow is a realistic workflow-based project that demonstrates how Next.js can be used to build production-style internal tools and management systems.
